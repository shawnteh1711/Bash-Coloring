# Bash-Coloring
Bash Coloring

## Basic Syntax
echo -e "\e[COLORmSample Text\e[0m"

Option	Description
-e	Enable interpretation of backslash escapes
\e  Begin the color modifications\
COLORm	Color Code + ‘m’ at the end
\e[0m   End the color modifications
